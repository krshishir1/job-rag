## Project

This is a Nodejs + Express application using Javascript.

## Package Manager

Use yarn only.
Never use npm or pnpm.

## Development

Run:

yarn dev

## Tests

Before finishing any task, run:

yarn test

If tests fail, fix them.

## Linting

Always run:

yarn lint

## Code Style

- Use modules instead of Common JS.
- Use async/await instead of .then().

## File Structure

API routes go inside:
src/routes

server.js -> main file
src/spec -> All spec files

## Git

Do not commit unless explicitly asked.
